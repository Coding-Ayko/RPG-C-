    <h1>Projeto Jogo de Aventura - Estruturas de Dados em C++</h1>
    <p>Este projeto foi criado para aprimorar as habilidades em Estruturas de Dados em C++, implementando um jogo de aventura com diferentes classes e interações.</p>
    
    <h2>Objetivo do Jogo</h2>
    <p>Explorar um mapa, interagir com diferentes elementos e inimigos, gerenciar um inventário de itens e tomar decisões estratégicas para sobreviver.</p>
    
    <h2>Instruções para Rodar o Jogo</h2>
    <ol>
        <li>Clone o repositório:</li>
        <pre><code>git clone https://github.com/seu-usuario/projeto-jogo-aventura.git</code></pre>
        <li>Compile os arquivos fontes do jogo:</li>
        <pre><code>g++ -o programa mapa.cpp personagem.cpp elemento.cpp inventario.cpp inimigo.cpp cgi.cpp driver.cpp</code></pre>
        <li>Execute o programa:</li>
        <pre><code>./programa.exe</code></pre>
    </ol>
    
    <h2>Instruções do Jogo</h2>
    <ul>
        <li><b>Explorar:</b> Movimente-se pelo mapa e descubra novos locais e itens.</li>
        <li><b>Inventário:</b> Consulte e gerencie os itens coletados.</li>
        <li><b>Lutar:</b> Enfrente inimigos usando armas e itens.</li>
        <li><b>Missões:</b> Complete objetivos para avançar na história.</li>
    </ul>
    
    <h2>Controles</h2>
    <ul>
        <li><b>Movimentação:</b> WASD ou setas.</li>
        <li><b>Interagir:</b> Pressione <code>E</code>.</li>
        <li><b>Atacar:</b> Pressione <code>A</code>.</li>
        <li><b>Abrir Inventário:</b> Pressione <code>I</code>.</li>
    </ul>
    
    <h2>Tecnologias Utilizadas</h2>
    <p><b>Linguagem:</b> C++</p>
    <p><b>Compilador:</b> GCC/Clang</p>
    <p><b>Bibliotecas:</b> Nenhuma biblioteca externa.</p>
